## Technical SEO Checklist

### Must be implemented before going live

- [ ] `sitemap.xml` — auto-generated by Wagtail at `/sitemap.xml`
- [ ] `robots.txt` — served by Django, disallows `/cms/` and `/admin/`
- [ ] Canonical URLs on every page (`<link rel="canonical">`)
- [ ] `hreflang` — not needed now (English only), add if Bengali version added later
- [ ] All images have `alt` attributes (populated from Wagtail's image `title`)
- [ ] No broken internal links
- [ ] Mobile-first responsive layout (Tailwind handles this)
- [ ] Core Web Vitals:
  - LCP < 2.5s (largest image preloaded with `priority` on `next/image`)
  - CLS < 0.1 (all images have explicit `width` and `height`)
  - FID/INP < 200ms (minimal JS, Server Components)

## Per-Page Metadata (Next.js `generateMetadata`)

Every page must export a `generateMetadata()` function. Pull values from the CMS API.

**Template:**
```ts
export async function generateMetadata({ params }): Promise<Metadata> {
  const data = await getProject(params.slug)
  return {
    title: data.seo_title || data.title,
    description: data.seo_description || data.short_description,
    openGraph: {
      title: data.seo_title || data.title,
      description: data.seo_description || data.short_description,
      images: [{ url: data.seo_og_image?.url || settings.og_image.url }],
      type: 'website',
    },
    twitter: {
      card: 'summary_large_image',
      creator: '@XRezwoan',
    },
    alternates: {
      canonical: `https://rezwoan.me/projects/${params.slug}`,
    },
  }
}
```

**Fallback chain:** `seo_title` → `title` → site default. Never leave a page with no title.

Every blog post and project page gets a unique OG image pulled from CMS data — this is what
makes social shares look professional and clickable.

Template: Dark background + project/post title in large bold type + Rezwoan branding.
